Why not just use a proprietary voice chat app like Dicksword?

Mumble just works and is easier/more ethical to use from an end-user standpoint. Everyone has more control on what data gets collected and send around, a lot of that has to do with the fact that Mumble is freedom software and a lot of the time -- there is zero need to collect "usage statistics" on a voice chat application. Discord continually has to come up with more ways to be "profitable" to justify continuing to operate as it currently does because from the ground up, Discord is designed to be anti-user/exploitative.

In short, Discord is a proprietary, bloated, user-unfriendly app that collects data on and tracks users. What we already know about Discord's TOS, moderation, and technical practices are horrific enough -- what we don't know what the program does (especially since application's source code is a secret/black box, which means we can't know exactly what the program is doing on your PC) is much worse. Here's a cute list that looks flashy for people with shorter attention spans:

  1. Discord's source code is secret

    It's impossible to download, examine, and verify what Discord as an application is doing because source code is properietary i.e. secret-source. Source code, by the way, is the human-readable set of instructions written by programmers using a programming language to command a computer to do things. Discord being proprietary software means that it is impossible to prove that Discord is not spyware: because we don't know what the code actually does. Any program which does not make its source code available is potential spyware.

  2. Discord explicitly confirms in its privacy policy that it collects and retains the following information from you:

    • IP Addresses
      • Discord can track your general location.
    • Device UUID (Universally Unique IDentifier)
      • Discord can tell what devices you use, how much you use each device, device usage habits.
    • Usage statistics surrounding ever other program on your PC
        • Discord uses a process logger to keep logs about what programs are open on your computer using a process logger
    • Email address
    • All messages sent on the platform, including images, DMs, voice messages, audio files, etc.
      • Discord logs every interaction you have with other users through its service: every conversation you have, everything you say, all images that you have sent. Nothing is private.
    • All VOIP (voice chat/recording) data
    • Open rates for e-mail sent by Discord
    • Other stuff that we don't know about... because the source code is a closed secret
      • Discord's privacy policy also contains several occurrences of phrases such as "including but not limited to," which is an explicit confirmation that Discord contains more spyware features that are not disclosed to the user.

  3. Discord Tries To Force Users To Give Phone Numbers

    Users are, after a time, eventually forced to provide their telephone numbers as a prerequisite for continued access or new registration. The criteria for triggering this demand isn't given (nor clearly defined or consistent), with some users experiencing it randomly, while others consistently encounter it when connecting through a VPN or Tor. This practice raises significant privacy concerns, as phone numbers are highly personal information that can be linked to various other personal details and potentially enable the identification and localization of users, posing security vulnerabilities like identity theft.

  4. Discord Readily Gives Government Agencies Information On Their Own Citizens

    Discord readily cooperates and complies with law enforcement and government officials to collect conversational information on their private citizens via a streamlined, institutionalized, and easy process they created with what Discord calls the Government Request Portal, a summary of what information shared includes:
    • User accout information
      • Email address
      • Phone number
      • IP address
      • Other account details not specified.
    • Message content
      • Includes private messages
    • Voice/video call logs
      • May provide logs of voice and video calls. Even though Discord claims that calls on the website are end-to-end encrypted, Discord's Developer Documentation reveals that Discord records metadata on calls, particularly: what & when calls were made, and who participated in said call to build circumstantial evidence against users regarding the content/context of said calls.
